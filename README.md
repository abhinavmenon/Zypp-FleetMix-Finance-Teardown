# Zypp Electric Finance Teardown

Independent finance and business analytics teardown of Zypp Electric using publicly available FY25 information and clearly identified assumptions.

## Objective

To explore how fleet mix, rider economics, and delivery utilization affect contribution margins and EBITDA performance.

## Key Questions

* Can increasing quick-commerce utilization improve profitability?
* How sensitive is EBITDA to rider payouts and commission rates?
* What operational KPIs should management monitor?

## Tools Used

* Financial Modelling
* Business Analytics
* HTML
* CSS
* JavaScript

## Disclaimer

This project is an independent educational analysis. Publicly reported information has been supplemented with clearly stated assumptions for modelling purposes and does not represent internal company data.
